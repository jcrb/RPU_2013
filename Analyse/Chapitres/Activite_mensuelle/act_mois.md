Activité par jour de semaine
----------------------------

```r
s <- wday(d1$ENTREE)
sf <- s
sf[s == "1"] <- 7
sf[s == "2"] <- 1
sf[s == "3"] <- 2
sf[s == "4"] <- 3
sf[s == "5"] <- 4
sf[s == "6"] <- 5
sf[s == "7"] <- 6

ts <- table(sf)
names(ts) <- semaine_f
```
